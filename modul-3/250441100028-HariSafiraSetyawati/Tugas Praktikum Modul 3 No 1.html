Tentang Saya

Saya adalah mahasiswa Sistem Informasi yang memiliki ketertarikan besar di bidang teknologi, khususnya web development dan desain grafis. Saya senang mempelajari hal-hal baru dan terus mengembangkan kemampuan dalam membuat website yang tidak hanya menarik secara visual, tetapi juga responsif dan mudah digunakan oleh pengguna. Bagi saya, proses membuat website merupakan hal yang menyenangkan karena saya bisa menuangkan ide dan kreativitas menjadi sebuah karya nyata yang bermanfaat. Selain itu, saya juga mencoba terbiasa menggunakan berbagai teknologi dasar seperti HTML, CSS, dan Python dalam mengembangkan website sederhana. Saya memiliki semangat belajar yang tinggi, mampu bekerja secara mandiri maupun dalam tim, serta bertanggung jawab dalam menyelesaikan setiap tugas yang diberikan. Ke depannya, saya ingin terus meningkatkan kemampuan di bidang pengembangan web dan mengikuti perkembangan teknologi agar dapat menciptakan karya yang lebih inovatif, profesional, dan berguna bagi banyak orang.

Keahlian

HTML

HTML digunakan untuk membuat struktur dasar sebuah website. Dengan HTML, saya dapat menyusun elemen seperti teks, gambar, link, dan bagian-bagian halaman agar tertata dengan rapi dan mudah dipahami oleh browser.

CSS

CSS digunakan untuk mengatur tampilan dan desain website agar lebih menarik. Saya menggunakan CSS untuk mengatur warna, layout, ukuran, serta membuat tampilan website menjadi responsif dan nyaman dilihat.

Python

Python adalah bahasa pemrograman yang saya gunakan untuk belajar logika pemrograman dan membuat program sederhana. Python juga dapat digunakan dalam pengolahan data, otomatisasi, dan pengembangan aplikasi.

Pengalaman

Membuat Website Cafe

Mengembangkan sebuah website landing page untuk produk makanan dan minuman dengan menggunakan HTML dan CSS. Dalam project ini, saya merancang tampilan yang menarik serta mengatur layout agar responsif di berbagai ukuran layar. Saya juga memperhatikan aspek-aspek tertentu agar website mudah digunakan dan nyaman dilihat oleh pengguna.

Project Playlist Musik

Membuat sistem sederhana berbasis Python untuk mengelola data playlist musik menggunakan konsep CRUD (Create, Read, Update, Delete). Dalam project ini, saya belajar mengelola data, memahami alur logika program, serta mengimplementasikan fungsi-fungsi dasar dalam pengolahan data.
